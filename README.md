# SkillPay AI Server

Deploy op Railway.app:

1. Ga naar railway.app en log in met GitHub
2. Klik "New Project" → "Deploy from GitHub repo"
3. Upload deze bestanden naar een GitHub repository
4. Railway detecteert automatisch Node.js en start de server
5. Kopieer de gegenereerde URL (bijv. https://skillpay-ai.up.railway.app)
6. Zet die URL in je Netlify environment variable: RAILWAY_URL

Klaar!
